# Data

As mentioned, the weight matrices are edited such that the bias terms take up the final row instead of the first row. Note that while most of the files are in decimal, the file "test_input.mem" is in hexadecimal since it is used for the HDL implementation.

Aflat is the test matrix we used for testing. The first three elements belong to the output layer weights matrix, the next 16 to the hidden layer weights matrix, and the rest are for the input matrix.
